# BCS Malawi Revamp

Modern static website for BCS Malawi, focused on enterprise IT services, managed networks, catalogue presentation, and customer issue reporting.

## Project Structure

- index.html: Home page with hero, service highlights, and issue reporting section
- services.html: Products and services overview
- managed-network.html: Managed network offering details
- accessories.html: Catalogue page
- contact.html: Contact and lead capture page
- about.html: Company profile and history
- assets/css/styles.css: Global styling and responsive layout
- assets/js/main.js: Navigation, reveal animations, form handling, and issue tracking logic

## Ticketing and Query Tracking

The home page includes a Contact Us and Report an Issue section.

- Generates a reference number for each submitted issue in the format BCS-YYYYMMDD-####
- Stores reports in browser localStorage
- Allows status updates and query tracking by reference number

## Run Locally

This is a static site.

1. Open index.html in a browser
2. Or use any local static server for development
